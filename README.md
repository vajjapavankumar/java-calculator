# Simple Java Calculator

A console-based calculator application written in Java.  
Built as a mini project for practice and portfolio (Cognizant GenC).

## Features

- Takes two numbers as input from the user
- Supports:
  - Addition
  - Subtraction
  - Multiplication
  - Division (with divide-by-zero check)
- Simple text menu in the console

## Technologies Used

- Java
- VS Code
